# Sagrada Família — ShadowCast (Clipped)

A minimal ArcGIS Maps SDK for JavaScript (4.29) demo showing sun shadows around Sagrada Família in Barcelona using the ShadowCast widget, with a clipped local 3D scene.

## Live Demo
https://alimadahii.github.io/sagrada-shadowcast/

## Repository
https://github.com/alimadahii/sagrada-shadowcast

## What it does
- Loads OSM 3D Buildings (SceneLayer)
- Focuses on Sagrada Família
- Clips the scene to a ~1200m buffer
- Lets users explore shadows interactively (ShadowCast widget)

## How to run locally
Open `index.html` using a local web server (recommended), e.g. VSCode Live Server.

## Notes
- Some basemap/services may require an ArcGIS API key (recommended for public demos).
